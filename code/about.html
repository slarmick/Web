🚀 Функциональность ЛР-5

Основные возможности:

  • MySQL Database Integration - работа с реляционной базой данных
  • Docker Multi-Container Setup - Nginx, PHP-FPM, MySQL, Adminer
  • PDO Database Class - безопасная работа с БД через подготовленные запросы
  • CRUD Operations - создание, чтение, удаление записей
  • Data Migration - сохранение данных в БД и файл одновременно
  • Adminer Integration - веб-интерфейс для управления БД
  • AJAX Delete Functionality - удаление записей без перезагрузки страницы
  • Error Handling - обработка ошибок подключения к БД

Новые файлы ЛР-5:

  • Dockerfile - конфигурация PHP с MySQL расширениями
  • Database.php - класс для работы с PDO и подключения к БД
  • MasterClassRegistration.php - класс для CRUD операций с таблицей
  • db.php - подключение к базе данных MySQL
  • delete_registration.php - обработчик удаления записей через AJAX

Как использовать:

  1. Запустите проект: docker-compose up -d --build
  2. Откройте в браузере: http://localhost:8080
  3. Перейдите на форму регистрации
  4. Заполните и отправьте форму
  5. Данные сохранятся в:
    • 🗄️ Базу данных MySQL (основное хранилище)
    • 📄 Файл data.txt (для обратной совместимости)
  6. Просмотр данных:
    • 📊 Все записи из БД и файла на странице view.php
    • 🗑️ Удаление записей из БД через кнопки в таблице
    • ⚙️ Управление БД через Adminer: http://localhost:8081

База данных:

Таблица: master_class_registrations

Поля: id, name, birthdate, topic, format, materials, email, created_at, updated_at

Подключение к MySQL: порт 3307, база: lab5_db, пользователь: lab5_user

Adminer: http://localhost:8081 (сервер: db, БД: lab5_db)
