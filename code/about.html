🎯 О проекте

Этот проект представляет собой серию лабораторных работ по изучению веб-программирования, контейнеризации и серверных технологий. Каждая лабораторная работа добавляет новый функционал и технологии к проекту.

💻 Лабораторная работа №3 Новая!

Тема: Обработка данных формы на PHP с сохранением в сессии и файл

PHP 8.2 Сессии PHP Валидация данных Работа с файлами Формы HTML Docker Compose
  • Обработка данных формы на стороне сервера через PHP
  • Сохранение данных в сессии PHP для временного хранения
  • Сохранение данных в текстовый файл data.txt для постоянного хранения
  • Валидация данных на стороне сервера (обязательные поля, email, возраст 18+)
  • Вывод всех сохраненных данных на отдельной странице view.php
  • Вывод ошибок валидации на главной странице
  • Отображение последних введенных данных из сессии
  • Защита от XSS атак с помощью htmlspecialchars()

🚀 Функциональность ЛР-3

Основные возможности:

  • Process.php - обработка формы, валидация, сохранение в сессию и файл
  • View.php - просмотр всех записей из data.txt в виде таблицы
  • Index.php - отображение данных из сессии и ошибок валидации
  • Валидация - проверка обязательных полей, email, возраста 18+
  • Безопасность - защита от XSS, безопасная работа с файлами

Как использовать:

  1. Запустите проект: docker-compose up -d
  2. Откройте в браузере: http://localhost:3000
  3. Перейдите на форму регистрации
  4. Заполните и отправьте форму
  5. На главной странице увидите результат
  6. Для просмотра всех данных нажмите "Посмотреть все данные"
