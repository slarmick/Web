🎯 О проекте

Этот проект представляет собой серию лабораторных работ по изучению веб-программирования, контейнеризации и серверных технологий. Каждая лабораторная работа добавляет новый функционал и технологии к проекту.

🔴 Лабораторная работа №6 Завершена

Тема: Нереляционные базы данных: Redis, Elasticsearch, ClickHouse

Redis 7 Elasticsearch 8.10 ClickHouse 24 Guzzle HTTP PHP Sessions Analytics Search Engine Docker Compose
  • Миграция сессий PHP из файловой системы в Redis для высокой производительности
  • Индексация данных пользователей в Elasticsearch для полнотекстового поиска
  • Сбор аналитики регистраций в ClickHouse для быстрой агрегации больших данных
  • Создание дашборда аналитики с статистикой в реальном времени
  • Реализация поиска по пользователям через Elasticsearch REST API
  • Разработка многокомпонентной архитектуры с graceful degradation
  • Настройка Docker Compose с 4 различными базами данных
  • Создание сервисов AnalyticsService и SearchService для работы с NoSQL
  • Визуализация статуса всех компонентов системы на главной странице
  • Обработка ошибок при недоступности отдельных компонентов системы

🚀 Функциональность ЛР-6

Основные возможности:

  • Redis Sessions - высокопроизводительное хранение сессий
  • Elasticsearch Search - полнотекстовый поиск по пользователям
  • ClickHouse Analytics - аналитика в реальном времени
  • Multi-Database Architecture - работа с 4 различными СУБД
  • Graceful Degradation - система работает даже при недоступности компонентов
  • Real-time Dashboard - дашборд с актуальной статистикой
  • Docker Orchestration - управление 5 контейнерами через Docker Compose

Новые файлы ЛР-6:

  • AnalyticsService.php - работа с ClickHouse для аналитики
  • SearchService.php - работа с Elasticsearch для поиска
  • analytics.php - дашборд аналитики
  • Обновленный Dockerfile - с расширением Redis
  • Обновленный docker-compose.yml - с 4 базами данных

Как использовать:

  1. Запустите проект: docker-compose up -d --build
  2. Откройте в браузере: http://localhost:8080
  3. Перейдите на форму регистрации и заполните данные
  4. Данные сохранятся в:
    • 🗄️ MySQL (основное хранение)
    • 🔴 Redis (сессии)
    • 🔍 Elasticsearch (поисковый индекс)
    • ⚡ ClickHouse (аналитика)
    • 📄 Файл data.txt (совместимость)
  5. Просмотр возможностей:
    • 📊 Все данные с поиском - view.php
    • 📈 Аналитика и графики - analytics.php
    • ⚙️ Управление БД - Adminer:8081
    • 🔍 Поиск по пользователям - через Elasticsearch

Доступ к системам:

Веб-сайт: http://localhost:8080

Adminer (MySQL): http://localhost:8081

Elasticsearch: http://localhost:9200

ClickHouse: порт 8123 (HTTP), 9000 (Native)

Redis: порт 6379
