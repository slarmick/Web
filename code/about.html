🚀 Функциональность ЛР-4

Основные возможности:

  • Composer Integration - управление зависимостями через composer.json
  • ApiClient Class - работа с Art Institute of Chicago API
  • UserInfo Class - сбор информации о браузере, IP, времени
  • Cookies - отслеживание времени последней отправки формы
  • Art Display - показ художественных произведений после отправки формы
  • Error Handling - обработка ошибок API с graceful degradation

Новые файлы ЛР-4:

  • ApiClient.php - класс для получения данных об искусстве
  • UserInfo.php - класс для информации о пользователе
  • composer.json - конфигурация зависимостей (Guzzle HTTP)
  • vendor/ - установленные пакеты Composer

Как использовать:

  1. Запустите проект: docker-compose up -d
  2. Откройте в браузере: http://localhost:3000
  3. Перейдите на форму регистрации
  4. Заполните и отправьте форму
  5. На главной странице увидите:
    • ✅ Данные формы
    • 👤 Информацию о вашем посещении
    • 🎨 Примеры художественных техник из музея
  6. Кука last_submission сохранит время отправки

API Endpoints:

Основной запрос: https://api.artic.edu/api/v1/artworks?limit=10&fields=id,title,artist_display,date_display,medium_display,image_id

Изображения: https://www.artic.edu/iiif/2/{image_id}/full/200,/0/default.jpg
